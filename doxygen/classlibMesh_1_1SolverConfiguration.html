Detailed Description

This class stores solver configuration data, e.g. tolerances and iteration limits. Override the pure virtual function configure_solver() in order to provide specific behavior for particular types of solvers (e.g. LinearSolver, EigenSolver, etc).

This method can be called after the solver has failed (e.g. on catching an exception thrown by a solver).
